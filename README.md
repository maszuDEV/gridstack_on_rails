# GridstackOnRails

Gridstack.js version: **1.2.1**

_Compatible with older browser like IE11..._

# Installation

Add this lines to your application's Gemfile:

```ruby
# To add jquery-ui
gem 'jquery-ui-rails'

# and gridstack
gem 'gridstack_on_rails', github: 'maszuDev/gridstack_on_rails'
```

And then execute:

```bash
bundle
```

And restart your server
